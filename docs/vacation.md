# Отпуск

## Основные правила

Сотруднику предоставляется 28 календарных дней отпуска в год.

Заявление на отпуск нужно подать минимум за 14 дней до начала отпуска.

Отпуск должен быть согласован с руководителем команды.

## Частые вопросы

Можно ли взять отпуск на один день?

Да, можно взять один день отпуска, если это согласовано с руководителем.

Можно ли перенести отпуск?

Да, перенос отпуска возможен по согласованию с руководителем и HR.
