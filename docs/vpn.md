# VPN

## Основные правила

VPN нужен для удалённого доступа к внутренним сервисам компании.

Доступ к VPN выдаётся только сотрудникам, которым он нужен для работы.

Для подключения VPN нужно использовать корпоративную учётную запись.

## Как получить доступ

Чтобы получить доступ к VPN, нужно отправить заявку в IT-отдел.

В заявке нужно указать имя сотрудника, отдел и причину необходимости VPN.

Обычно заявка обрабатывается в течение 1–2 рабочих дней.

## Контакты

По вопросам VPN нужно обращаться в IT-отдел.
